# Intelligent Travel Assistant AI

## Overview
This project is an AI-powered Travel Assistant built using LangChain Tool Calling Agents.

The assistant helps users:
- Get current weather information for a destination.
- Discover top tourist attractions in a city.
- Receive a combined travel recommendation generated by an LLM.

## Features
- Custom Weather Tool using WeatherAPI
- Tourist Attraction Search using DuckDuckGo
- LangChain Tool Calling Agent
- LLM-based reasoning and response generation

## Technologies Used
- Python
- LangChain
- OpenAI GPT
- WeatherAPI
- DuckDuckGo Search

## Project Workflow

User Input
→ LangChain Agent
→ Weather Tool
→ Attractions Search Tool
→ LLM Reasoning
→ Final Travel Recommendation

- travel_assistant_agent.py : Main application
